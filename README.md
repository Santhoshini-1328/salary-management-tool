# Salary Management Tool

## Overview

This repository contains a salary management tool with a TypeScript backend and a React frontend.

### Features completed
- Server-side employee grid with search and pagination
- Add, view, edit, and delete employees via UI
- Employee fields include full name, email, country, job title, department, and salary
- Salary insights page with country salary metrics and job-title benchmarks
- Dashboard with total employee count, average salary, and highest paying roles
- Backend pagination support and frontend server-side grid fetching
- Backend Jest test covering employee pagination service

## Run locally

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend tests
```bash
cd backend
npm test
```
